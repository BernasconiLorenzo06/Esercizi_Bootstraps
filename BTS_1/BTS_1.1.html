A cosa serve bootstrap?

Bootstrap è un framework front-end gratuito per uno sviluppo web più rapido e semplice

Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, oltre a plugin JavaScript opzionali

Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi

Cosa devi importare per creare una pagina web con bootstrap ?

Per importare il bootstrap il tag da scrivere è il seguente

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/css/bootstrap.min.css" rel="stylesheet">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Un Content Delivery Network (CDN) è un gruppo di server distribuiti progettati per accelerare la distribuzione dei contenuti.
I CDN non possono ospitare contenuti, quindi non sostituiscono il web host. Al contrario, un CDN può stabilizzare i siti web e aumentarne le prestazioni:

A cosa serve questo attributo del tag meta?

content="width=device-width, initial-scale=1>

indica al browser di adattare la larghezza della pagina alla larghezza del dispositivo e di impostare lo zoom iniziale a 1

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

The .container class provides a responsive fixed width container

The

conteiner-fluid

class provides a full width container, spanning the entire width of the viewport
